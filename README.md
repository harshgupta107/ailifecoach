# ✦ AI Life Coach

A personalized AI coaching assistant powered by Google Gemini 2.5 Flash.

## Features
- 🎯 Set goals across health, career, learning, mindset & finance
- 📝 Daily mood & progress check-ins
- 💬 Real-time AI coaching chat
- 📊 AI-generated progress summaries

## Setup
1. Open the notebook in Google Colab
2. Run Cell 1 & 2 to install dependencies
3. Add your Gemini API key in Cell 3
4. Run all cells and start chatting!

## Requirements
- Google Gemini API Key (free at https://aistudio.google.com)
- Google Colab or Jupyter Notebook
